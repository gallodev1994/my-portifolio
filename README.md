## 📝 Configuração do Formulário de Contato

O formulário de contato utiliza [EmailJS](https://www.emailjs.com/) para envio de e-mails.

1. Crie uma conta no [EmailJS](https://www.emailjs.com/)
2. Configure o serviço em `src/app/[lang]/components/contact/contact.tsx`
3. Crie um template com as variáveis:
   - `{{from_name}}`
   - `{{from_email}}`
   - `{{message}}`
   - `{{to_email}}`
